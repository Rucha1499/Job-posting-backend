## Job-posting-backend

### This repository contains the backend code for the job posting application. Hosted [here](https://job-posting-application.herokuapp.com/).

### Running the code
- Clone the repository.
- run `npm install` to install all the packages and dependencies.
- run `npm run dev` to run the code

### Scripts
- `npm run lint` runs eslint and checks all the code styles and formatting.
- `npm run lint-fix` fixes all the formatting and code style errors and warnings.
